## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: Minimum 1280x720 (HD)
- **File Size**: Keep under 500KB for optimal loading
- **Naming Convention**: Use the format `##-descriptive-name.png`
- **Content**: Ensure no sensitive data (API keys, subscription IDs) is visible

## How to Capture Screenshots

1. Navigate to Azure AI Foundry portal (ai.azure.com)
2. Follow the steps in the main README
3. Use your operating system's screenshot tool:
   - **Windows**: Windows Key + Shift + S
   - **macOS**: Cmd + Shift + 4
   - **Linux**: Print Screen or Screenshot tool

4. Save screenshots to this directory with the appropriate filename
5. Verify images are referenced correctly in the main README.md
